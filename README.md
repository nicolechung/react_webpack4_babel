# React setup with Webpack 4 and Babel 7

# Styles
Sass with just enough postcss to have autoprefixer

# Setup
Make sure you have a recent version of npm and node:

```
node --version
v8.12.0

npm --version
6.4.1
```

Babel 7 won't run on versions of node earlier than 8.

To install packages:
`npm install`

To run the project:
`npm start`
